Kali Linux


Last Update: 2025-03-19 18:14 UTC

  • OS Type: Linux
  • Based on: Debian (Testing)
  • Origin: Gibraltar
  • Architecture: aarch64, Apple M1, armel, x86_64
  • Desktop: GNOME, KDE Plasma, Xfce
  • Category: Data Rescue, Forensics, Live Medium, Raspberry Pi, Security
  • Status: Active
  • Popularity: 18 (417 hits per day)
Kali Linux (formerly known as BackTrack) is a Debian-based distribution with a collection of security and forensics tools. It features timely security updates, support for the ARM architecture, a choice of four popular desktop environments, and seamless upgrades to newer versions.

I had kali gnome on three different computers, the updates killed them all.
So on one laptop I reinstalled the march 19 iso.
It worked grreat until I installed the updates again.
Same issue,
Another thing that is weird about kali.
One iso for live testing, and one iso to install the system.
All the other big linux names have live, and install on same usb iso.
I can see having an update issue now and then?
But, to not be able to login at all is too big of an issue for me.
So I am abandoning a system I had great promise for.

 The best distro for PEN testing. Not easy for new comers like me, but if you don't start you won't learn. I had few issues with Kali, not because of the distro itself rather that the destructive commands I ran without paying attention to the consequences, but I got it back fully working. It has become my daily OS usage, I finally got rid of Windows which I'll never come back to use again EVER even if it becomes free!

Kali has got a good documentation page, especially for the tools included, not easy to use for beginners, but this OS isn't intended for beginners anyway, so no lamentation here. With Linux in general, you must insist in using it and learn your way through the Terminal commands, which is what I am doing. By the way, I am a professional IT guy coming from the Microsoft land. So I have no problems dealing with hard tasks from another OS, it only needs patience, some research, and insisting on learning, to find out that Linux Rocks and it really a lightweight OS with respect to the bloated Windows/macOS. My choice was Kali simply because I come from the Security team on Windows land, and I couldn't be happier with Kali! The learning curve is steep, but the satisfaction is great.
