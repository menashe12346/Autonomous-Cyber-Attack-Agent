Super Grub2 Disk

Super Grub2 Disk is a live CD that helps the user to boot into almost any operating system even if the system cannot boot into it by normal means. This allows a user to boot into an installed operating system if their GRUB installation has been overwritten, erased or otherwise corrupted. Super Grub2 Disk can detect installed operating systems and provide a boot menu which allows the user to boot into their desired operating system. Super Grub2 Disk is not an operating system itself, but a live boot loader which can be run from a CD or USB thumb drive.

 This can be a lifesaver, and would be a good tool for any system administrator to keep at hand.

I routinely use Debian Unstable. Yesterday an update to GRUB had a bug, and my system wouldn't boot. Fortunately everything else was intact. The usual advice is to boot using an installation image, and use that to re-install Grub2, but this didn't work for some reason. I was able to boot my system with the Super Grub2 image, then apply a new update that fixed the bug with GRUB.

In the past, I've used this when sorting out issues with multiple operating systems on physical partitions.
